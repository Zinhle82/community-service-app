# Community Service Issue Reporting System

## Description

This project is a Django-based web application that allows users to report community service issues and track their status.

## Features

* Submit community issues
* View reported issues
* Simple and user-friendly interface

## Technologies Used

* Python
* Django
* HTML

## How to Run the Project

1. Activate the virtual environment
2. Run the command: python manage.py runserver
3. Open a browser and go to http://127.0.0.1:8000/

## Project Structure

* reports: Handles issue submission
* tracking: Displays reported issues
* templates: Contains HTML pages
